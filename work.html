広告ターゲティングモデル構築（LINEヤフー在籍時）

役割：MLエンジニア／データサイエンティスト
体制：要件整理・特徴量設計・モデル学習・A/Bテスト・本番運用

背景・課題

大規模ユーザーデータに基づき、広告配信のCTR/CVRの最適化が求められていた。
配信効果を最大化するため、ターゲティング精度向上と運用しやすいML基盤の整備が必要だった。

アプローチ

  • LLMを活用して行動ログから多様な特徴量を設計
  • Pytorch／TensorFlowによるベクトル表現学習・分類学習
  • Spark／Hive によるデータパイプライン、Airflowでスケジューリング
  • オンラインA/Bテストで効果検証

成果

  • 再現性の高いMLパイプラインを整備し、後続施策の展開を加速
Python TensorFlow PyTorch Spark Hive Airflow Ranking A/B Testing

レコメンドシステム開発・最適化（リクルート在籍時）

役割：MLエンジニア／データサイエンティスト
体制：要件整理・特徴量設計・モデル学習・API開発・A/Bテスト・本番運用

背景・課題

大規模なユーザー×アイテム環境で、レコメンド精度と事業KPI（資料請求）の向上が求められていた。
オンライン配信の可用性を保ちつつ、予測精度向上が課題。

アプローチ

  • BigQuery に蓄積されたユーザ行動ログとアイテム情報を集計し、特徴量を抽出
  • LightGBM のランキング学習（LGBM Ranker）を用いたレコメンドモデルを構築
  • モデルの推論APIを AWS ECS 上に実装し、本番配信に統合
  • オンラインA/Bテストにより効果検証を行い、継続的に改善

成果

  • KPIを改善に貢献
  • 再現性の高いMLパイプラインを整備し、後続の機能追加・施策展開を加速
Python LightGBM BigQuery Airflow Recommendation Systems Ranking A/B Testing
